# 💬 Pop-Up Message Project

A simple and interactive pop-up message box built using HTML and CSS.
This project demonstrates how to create a modal/pop-up without using JavaScript.

## 📌 Features
* 💬 Pop-up message box
* 🎨 Clean and minimal UI
* ✨ Smooth transition effect
* 🔗 CSS `:target` selector functionality
* 📱 Beginner-friendly responsive design


## 🛠️ Technologies Used
* HTML5
* CSS3

## 💡 How It Works

The pop-up appears when the button is clicked using the CSS `:target` pseudo-class.

```css id="eh0x2v"
.popup:target{
    visibility: visible;
    opacity: 1;
}

This allows the modal to open without JavaScript.
